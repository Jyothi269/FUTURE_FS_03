# Jyothi Sweet Treat Bakery Website

## Project Pitch

I created a professional website for “Jyothi Sweet Treat Bakery”, a local bakery business.

The main purpose of this website is to help the bakery attract more customers by improving its online presence and making it easier for people to view their menu, location, and contact details.

The website includes sections like Home, About, Menu, Gallery, Testimonials, FAQ, and Contact.

It also includes features like WhatsApp ordering, Google Maps integration, and responsive design.

This project helps improve customer trust and increases online visibility for the business.
